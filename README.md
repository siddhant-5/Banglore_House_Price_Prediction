# 🏠 Bangalore House Price Prediction

A simple machine learning web app that predicts house prices in Bangalore based on property features.

## 🚀 Live Demo

👉 https://your-username-bangalore-house-price-prediction.streamlit.app

---

## 📌 Features

* Predict house prices instantly
* User-friendly interface with Streamlit
* Lightweight and fast ML model

---

## 🛠️ Tech Stack

* Python
* Streamlit
* Scikit-learn
* NumPy / Pandas

---

## ⚙️ Run Locally

```bash
git clone https://github.com/siddhant-5/Banglore_House_Price_Prediction.git
cd Banglore_House_Price_Prediction
pip install -r requirements.txt
streamlit run app.py
```
